# UX/UI Redesign for TB Group Corporate Website

## Product Overview
Transform the TB Group corporate website from a standard business site into a visually stunning, highly interactive experience using modern design trends (2025). The goal is to increase user engagement by 40% and conversion rate by 25% through exceptional UX/UI.

## Current State Analysis
**Strengths:**
- Next.js 14 + TypeScript architecture
- Dark theme (slate-950)
- Framer Motion animations
- Responsive design
- Docker containerization

**Pain Points:**
- Static, non-interactive components
- Limited visual hierarchy
- No micro-interactions
- Outdated typography
- Weak CTA visibility
- Missing iconography
- No particle effects or 3D elements

## Target Users
1. **Business Owners** - Seeking cloud solutions (45-60 years)
2. **IT Managers** - Evaluating technology partners (30-50 years)
3. **Operations Managers** - Looking for process automation (25-45 years)

## User Experience Goals
1. **First Impression** - Wow factor within 3 seconds
2. **Navigation** - Intuitive and smooth
3. **Content Consumption** - Engaging and scannable
4. **Conversion** - Clear path to contact/quote
5. **Performance** - <2s load time

## Appendix

### A. Animation Guidelines
- Keep animations under 300ms
- Use transforms for GPU acceleration
- Respect `prefers-reduced-motion`
- Ensure 60fps performance
- Test on 60Hz and 120Hz displays

### B. Color Usage
- Primary: 60% of UI
- Secondary: 30% of UI
- Accent: 10% of UI
- Maintain consistent contrast ratios

### C. Typography Rules
- Maximum 8 font sizes
- Clear hierarchy (H1-H6)
- Line height: 1.5-1.75 for body
- Line length: 45-75 characters

### D. Icon System
- Use consistent style (outline vs filled)
- Size variants: 16px, 20px, 24px, 32px
- Always include aria-label
- Use for navigation, actions, status

### E. Form Guidelines
- Clear labels
- Helpful placeholders
- Real-time validation
- Accessible error messages
- Success confirmation
